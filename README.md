# and
젤라또 브랜드 브랜딩 및 퍼블리싱

## 📃요약
- 젤라또 판매점 브랜딩 디자인 + 웹 퍼블리싱 프로젝트입니다
- 디자인과 프론트 화면 구현에 집중하였습니다
- 원페이지로 제작했습니다
- PC / 태블릿 / 모바일에 맞추어 반응형 페이지로 제작했습니다
- jQuery를 이용하여 페이지 조작 기능을 구현했습니다
    - 메뉴 클릭하면 각각의 상세메뉴 팝업창 노출
    - 창업소개 / 창업약속 탭메뉴 구현

## 🛠사용 기술 및 라이브러리
`html` `css` `jQuery`
